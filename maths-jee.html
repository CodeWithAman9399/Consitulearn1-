Chapter 1 - Sets, Relations & Functions

1. Sets, Operations, and Power Set

i. Sets

A set is a collection of distinct objects, called elements, written inside {}.

Types of Sets:

  • Empty Set (∅): A set with no elements.
  • Finite & Infinite Sets: A set with a countable or uncountable number of elements.
  • Subset (⊆): A set A is a subset of B if all elements of A are in B.
  • Proper Subset (⊂): A subset that is not equal to the given set.
  • Power Set: The set of all subsets of a given set.
  • Universal Set (U): A set containing all elements under discussion.

ii. Operations on Sets

  • Union (∪): A set containing all elements from A and B.
    Example: If A = {1,2,3}, B = {3,4,5}, then A ∪ B = {1,2,3,4,5}
  • Intersection (∩): A set containing common elements of A and B.
    Example: A ∩ B = {3}
  • Difference (-): A set containing elements in A but not in B.
    Example: A - B = {1,2}
  • Complement (A’): Elements in the universal set but not in A.
  • Symmetric Difference (Δ): Elements in either A or B but not both.
    Example: A Δ B = {1,2,4,5}

iii. Power Set

Definition: The set of all subsets of a given set.

Formula: If A has n elements, then P(A) has 2^n subsets.

Example: If A = {1,2}, then P(A) = {∅, {1}, {2}, {1,2}}.

Properties:

  • Always includes {} and the set itself.
  • If |A| = n, then |P(A)| = 2^n.

2. Relations

A relation between two sets A and B is a subset of their Cartesian product A × B. It defines a relationship between elements of A and B.

Types of Relations

  • Empty Relation: No element of A is related to any element of B.
  • Universal Relation: Every element of A is related to every element of B.
  • Reflexive Relation: If (a, a) ∈ R for every a ∈ A.
  • Symmetric Relation: If (a, b) ∈ R implies (b, a) ∈ R.
  • Transitive Relation: If (a, b) ∈ R and (b, c) ∈ R, then (a, c) ∈ R.
  • Equivalence Relation: A relation that is reflexive, symmetric, and transitive.

Equivalence Relation

A relation R on a set A is an equivalence relation if it satisfies:

  • Reflexivity: (a, a) ∈ R for all a ∈ A
  • Symmetry: If (a, b) ∈ R, then (b, a) ∈ R
  • Transitivity: If (a, b) ∈ R and (b, c) ∈ R, then (a, c) ∈ R

Example:

Consider a relation R on set A = {1,2,3,4} defined by a R b if a mod 2 = b mod 2.

  • Reflexive: Every element is related to itself.
  • Symmetric: If a R b, then b R a.
  • Transitive: If a R b and b R c, then a R c.

Thus, R is an equivalence relation.

3. Functions: One-One, Onto, and Composition

A function is a special type of relation where each element of a set A (domain) is assigned exactly one element of a set B (codomain). It is denoted as:

f: A → B, where A is the domain, B is the codomain, and f(x) is the output for each x in A.

i. One-One (Injective) Function

A function is called one-one (injective) if different elements in A map to different elements in B. This means no two different inputs have the same output.

Mathematical Definition: If f(a1) = f(a2), then a1 = a2 (If outputs are the same, then inputs must also be the same).

Example:

Let f(x) = 2x + 3. If f(a1) = f(a2), then 2a1 + 3 = 2a2 + 3, which simplifies to a1 = a2. Since different values of x give different outputs, it is one-one.

Graphical Representation:

A function is one-one if any horizontal line cuts the graph at most once.

  • f(x) = x and f(x) = e^x are one-one.
  • f(x) = x² is not one-one because f(-x) = f(x).

ii. Onto (Surjective) Function

A function is called onto (surjective) if every element in the codomain B has at least one preimage in domain A. This means that the function covers all elements of B.

Mathematical Definition: For every b ∈ B, there exists an a ∈ A such that f(a) = b.

Example:

Let f(x) = x³ from ℝ → ℝ. Since every real number has a cube root, every y ∈ ℝ has a preimage x = y³. Thus, the function is onto.

Graphical Representation:

A function is onto if its graph covers the entire range of the codomain.

  • f(x) = x² from ℝ → ℝ is not onto since negative numbers are missing from the range.
  • f(x) = x³ is onto, as it covers all real numbers.

iii. One-One and Onto (Bijective) Function

A function is called bijective if it is both one-one and onto.

Properties of a Bijective Function:

  • It has a unique output for every input (one-one).
  • It covers the entire codomain (onto).
  • It has an inverse function f⁻¹ such that f⁻¹(f(x)) = x.

Example:

f(x) = x + 2 from ℝ → ℝ is bijective, since it is both injective and surjective.

f(x) = x² is not bijective because it is neither one-one nor onto.

iv. Composition of Functions

The composition of functions is applying one function after another. If we have two functions:

  • f: A → B
  • g: B → C

Then the composition is written as:

(g ∘ f)(x) = g(f(x)), meaning that we first apply f(x), then apply g(x) to the result.

Example of Function Composition:

Let:

  • f(x) = x + 2
  • g(x) = x²

Then, (g ∘ f)(x) = g(f(x)) = (x + 2)²

Properties of Function Composition:

  • Associativity: (h ∘ g) ∘ f = h ∘ (g ∘ f) — Applying functions in different orders gives the same result.
  • Identity Function: I(x) = x, if I(x) is the identity function, then f ∘ I = I ∘ f = f.
  • Inverse Function: If f: A → B and g: B → A such that g(f(x)) = x, then g is the inverse of f.

Example of Inverse Function:

If f(x) = 2x + 3, then solving for x gives x = (y - 3)/2, and thus f⁻¹(x) = (x - 3)/2.

Chapter 2 - Complex Numbers & Quadratic Equations

1. Complex Numbers, Argand Plane, Modulus & Argument

i. Complex Numbers

A complex number is a number of the form:

z = a + bi, where:

  • a is the real part (Re(z))
  • b is the imaginary part (Im(z))
  • i (called iota) is the imaginary unit, defined as i² = -1

Operations on Complex Numbers:

  • Addition: (a + bi) + (c + di) = (a + c) + (b + d)i
  • Subtraction: (a + bi) - (c + di) = (a - c) + (b - d)i
  • Multiplication: (a + bi) × (c + di) = ac - bd + (ad + bc)i (since i² = -1)
  • Division: (a + bi) / (c + di) = (a + bi)(c - di) / (c² + d²)

ii. Argand Plane (Complex Plane)

The Argand plane is a way to represent complex numbers graphically.

  • The x-axis represents the real part.
  • The y-axis represents the imaginary part.
  • A complex number z = a + bi is plotted as the point (a, b) in the plane.

Examples:

  • z = 3 + 4i is plotted at (3, 4)
  • z = -2 - i is plotted at (-2, -1)

iii. Modulus of a Complex Number

The modulus (absolute value) of a complex number z = a + bi is given by:

|z| = √(a² + b²)

It represents the distance from the origin in the Argand plane.

Example:

For z = 3 + 4i, the modulus is:

|z| = √(3² + 4²) = √(9 + 16) = √25 = 5

iv. Argument of a Complex Number

The argument (or phase) of a complex number is the angle θ it makes with the positive x-axis, given by:

θ = tan⁻¹(b / a)

  • Principal argument: -π < θ ≤ π
  • General argument: θ + 2kπ, k ∈ Z

Example:

For z = 1 + i, the argument is:

θ = tan⁻¹(1 / 1) = tan⁻¹(1) = π/4

v. Polar Form of a Complex Number

A complex number can also be written in polar form:

z = r (cos θ + i sin θ), where:

  • r = |z| is the modulus
  • θ is the argument

Example:

For z = 1 + i:

  • r = √(1² + 1²) = √2
  • θ = π/4

Thus, z = √2 (cos π/4 + i sin π/4)

2. Quadratic Equations, Roots & Coefficients

i. Quadratic Equation

A quadratic equation is a second-degree polynomial equation in a single variable, generally written in the form:

ax² + bx + c = 0, where:

  • a, b, and c are constants (real or complex numbers),
  • x is the variable,
  • a ≠ 0 (because if a = 0, it becomes a linear equation).

ii. Roots of a Quadratic Equation

The roots of a quadratic equation are the values of x that satisfy the equation ax² + bx + c = 0. These can be found using various methods:

  • Factoring (when possible),
  • Completing the square,
  • Quadratic Formula: This is the most widely used method to find the roots, and it is derived from completing the square. The quadratic formula is given by:

x = (-b ± √(b² - 4ac)) / 2a

Where:

  • b² - 4ac is called the discriminant.

iii. Discriminant

The discriminant (Δ) is the expression b² - 4ac under the square root in the quadratic formula. It provides information about the nature of the roots of the quadratic equation:

  • If Δ > 0: The equation has two real and distinct roots.
  • If Δ = 0: The equation has two real and identical roots (or one real root).
  • If Δ < 0: The equation has no real roots, but two complex roots.

iv. Relationship Between Roots and Coefficients (Vieta's Formula)

For a quadratic equation ax² + bx + c = 0, the following relationships exist between the roots and coefficients:

  • Let the roots be r₁ and r₂. From Vieta's formulas:
  • Sum of the roots: r₁ + r₂ = -b / a
  • Product of the roots: r₁ ⋅ r₂ = c / a

v. Coefficients

In a quadratic equation ax² + bx + c = 0:

  • a is the coefficient of x², determining the parabola's "width" and "direction."
  • b is the coefficient of x, affecting the slope and position of the parabola.
  • c is the constant term, determining the vertical position of the parabola.

vi. Properties of Quadratic Equations

The graph of a quadratic equation is a parabola. The following are key properties:

  • If a > 0, the parabola opens upward, and if a < 0, it opens downward.
  • The vertex of the parabola (the point where it reaches its maximum or minimum) occurs at:

x = -b / 2a

At this point, the value of y (or the value of the quadratic equation) is given by:

y = (4ac - b²) / 4a

vii. Example

Solve the quadratic equation 2x² - 4x - 6 = 0.

Identify coefficients:

  • a = 2, b = -4, c = -6

Find the discriminant:

Δ = b² - 4ac = (-4)² - 4(2)(-6) = 16 + 48 = 64

Find the roots using the quadratic formula:

x = (-(-4) ± √64) / (2(2)) = (4 ± 8) / 4

  • x = (4 + 8) / 4 = 12 / 4 = 3
  • x = (4 - 8) / 4 = -4 / 4 = -1

Thus, the roots are x = 3 and x = -1.

1. Types & Operations on Matrices

Definition

  • A matrix is a rectangular array of numbers arranged in rows and columns.
  • It is typically denoted by a capital letter, and its elements are denoted by lowercase letters with two subscripts: a ij , where i represents the row and j represents the column.

Types of Matrices

  1. Row Matrix

    • A matrix with only one row, i.e., it is of the form 1 ×n .
    • Example: a 11 a 12 a 13 a 1n

  2. Column Matrix

    • A matrix with only one column, i.e., it is of the form m ×1 .
    • Example: a 11 a 21 a 31 ⋮ a m1

  3. Square Matrix

    • A matrix where the number of rows equals the number of columns (e.g., n ×n ).
    • Example: a 11 a 12 ⋮ a n1

  4. Zero (Null) Matrix

    • A matrix in which all elements are zero. It can be of any size.
    • Example: 0 0 0 0

  5. Identity Matrix

    • A square matrix where all the diagonal elements are 1, and all other elements are 0.
    • Example: 1 0 0 1 0 0

Operations on Matrices

Matrix Addition

  • Two matrices can be added if and only if they have the same dimensions.
  • Formula: C ij = A ij + B ij

Matrix Subtraction

  • Matrix subtraction is similar to addition, only possible when matrices have the same dimensions.
  • Formula: C ij = A ij - B ij

Scalar Multiplication

  • A matrix can be multiplied by a scalar.
  • Formula: B ij = k ⋅ A ij

  • Matrix multiplication is only possible if the number of columns of the first matrix equals the number of rows of the second matrix.

  • For a square matrix, the determinant is often denoted as det ( A ) or | A | .

ii. Determinant of a 2x2 Matrix

The determinant of A is calculated as:

det ( A ) = a ( e i - f h ) - b ( d i - f g ) + c ( d h - e g )

  • Determinant of a Diagonal or Scalar Matrix: The determinant of a diagonal matrix (or scalar matrix) is the product of the diagonal elements. For example, for a diagonal matrix: A 2×2 = a 11 a 22 ⇒ det ( A ) = a ⋅ b
  • Determinant of the Identity Matrix: The determinant of the identity matrix is 1. det ( I ) = 1
  • Determinant of a Zero Matrix: The determinant of a zero matrix is always 0. det ( 0 ) = 0
  • Row Interchanges: If two rows (or two columns) of a matrix are interchanged, the sign of the determinant changes (it gets multiplied by -1).
  • Determinant of a Matrix Product: The determinant of the product of two matrices is the product of their determinants: det ( A B ) = det ( A ) ⋅ det ( B )
  • Scalar Multiplication: If each element of a row (or column) of a matrix is multiplied by a scalar k , the determinant of the matrix is multiplied by k raised to the power of n for an n × n matrix: det ( kA ) = k ⁿ ⋅ det ( A )
  • Singular Matrix: A matrix is singular (non-invertible) if and only if its determinant is 0. That is, det ( A ) = 0 .

3. Inverse of a Matrix & Solving Linear Equations

Definition

  • The inverse of a matrix \( A \) is a matrix \( A^{-1} \) such that when multiplied with \( A \), it results in the identity matrix \( I \).
  • Formula: \( A \times A^{-1} = A^{-1} \times A = I \)

Conditions for Invertibility

  • The matrix \( A \) has an inverse if and only if \( \det(A) \neq 0 \).
  • If \( \det(A) = 0 \), the matrix is called singular and does not have an inverse.

Inverse of a 2x2 Matrix

  • For a 2x2 matrix:
    • A = a b c d
  • The inverse of matrix \( A \), denoted as \( A^{-1} \), is given by:
    • A -1 = 1 det(A) d -b -c a

Example of Inverse of a 2x2 Matrix

  • Given the matrix:
    • A = 2 3 1 4
  • The determinant of \( A \) is:
    • det(A) = (2)(4) - (3)(1) = 5
  • Since \( \det(A) = 5 \neq 0 \), the inverse exists:
    • A -1 = 1 5 4 -3 -1 2

Solving Linear Equations Using the Inverse Matrix

  • Consider the system of linear equations:
    • a 1 x + b 1 y = c 1 a 2 x + b 2 y = c 2
  • This can be written as a matrix equation:
    • A · X = C
  • To solve for \( X \), we multiply both sides by \( A^{-1} \):
    • A -1 · A · X = A -1 · C
  • Since \( A^{-1} \cdot A = I \), we get:
    • X = A -1 · C

Example: Solving a System of Equations

  • Given the system:
    • 2x + 3y = 5 x + 4y = 6
  • In matrix form:
    • A = 2 3 1 4
  • Multiplying the inverse of \( A \) by \( C \) gives:
    • X = A -1 · C

General Case for Solving Linear Equations

  • For a system of \( n \) linear equations in \( n \) variables:
    • A · X = C
  • The solution is given by:
    • X = A -1 · C
Chapter 4 - Permutations & Combinations

Counting Principle (Fundamental Principle of Counting)

Definition

  • The Counting Principle states that if there are m ways to perform one task and n ways to perform a second task, then the number of ways to perform both tasks is the product of the number of ways to perform each task.
  • Formula: Total Ways = m × n
  • This can be extended to multiple tasks. If there are m1, m2, ... , mk ways to do each task, the total number of ways to complete all tasks is:

    • Total Ways = m1 × m2 × ... × mk

Applications

  • Used in combinatorics to count possible outcomes or arrangements.
  • Helps determine the number of ways to combine various tasks or events.

P(n, r) - Permutations

Definition

  • A permutation refers to an arrangement or ordering of objects where the order matters.
  • P(n, r) represents the number of ways to arrange r objects from a set of n distinct objects, where the order of selection matters.
  • Formula: P ( n , r ) = n ! ( n − r ) r !

Example of Permutations (P(n, r))

Consider selecting and arranging 2 books from a collection of 5 distinct books.

  • n = 5, r = 2.

    • Using the formula:

    P(5, 2) = 5! / (5-2)! = 5 × 4 = 20

    So, there are 20 ways to arrange 2 books out of 5.

Applications

  • Used to calculate the number of ways to arrange items in specific orders.
  • Important in problems where the order of selection or arrangement matters (e.g., arranging people, seats, or objects).

C(n, r) - Combinations

Definition

  • A combination refers to the selection of objects where the order of selection does not matter.
  • C(n, r) represents the number of ways to select r objects from a set of n distinct objects, where the order of selection does not matter.
  • Formula: C ( n , r ) = n ! r ! ⁢ ( n − r ) !

Example of Combinations (C(n, r))

Consider selecting 2 books from a collection of 5 distinct books, where the order does not matter.

  • n = 5, r = 2.

    • Using the formula:

    C(5, 2) = 5! / (2! × (5-2)!) = 5 × 4 / (2 × 1) = 10

    So, there are 10 ways to select 2 books from 5 where order does not matter.

Applications

  • Used when order of selection is irrelevant (e.g., selecting members for a team).
  • Important in probability, where the order of outcomes does not affect the result (e.g., drawing cards from a deck).

Key Differences Between Permutations and Combinations

Permutations (P(n, r))

  • The order of selection matters.
  • For example, arranging 2 books from 5 distinct books is different from selecting them.

Combinations (C(n, r))

  • The order of selection does not matter.
  • For example, selecting 2 books from 5 distinct books is the same whether you pick book A first or book B first.

Permutations Example (P(n, r))

Problem: How many different ways can a committee of 3 members be arranged from a group of 5 people?

  • n = 5, r = 3.

    • Using the formula for permutations:

    P(5, 3) = 5! / (5-3)! = 5 × 4 × 3 = 60

    So, there are 60 ways to arrange the committee.

Combinations Example (C(n, r))

Problem: How many ways can a team of 3 members be selected from a group of 5 people (order doesn't matter)?

  • n = 5, r = 3.

    • Using the formula for combinations:

    C(5, 3) = 5! / (3! × (5-3)!) = 5 × 4 × 3 / (3 × 2 × 1) = 10

    So, there are 10 ways to select the team.

Summary

  • Counting Principle: A fundamental rule for counting the total number of outcomes or tasks by multiplying possibilities.
  • Permutations (P(n, r)): Arrangements where the order matters, calculated using the formula P(n, r) = n! / (n - r)!.
  • Combinations (C(n, r)): Selections where order does not matter, calculated using the formula C(n, r) = n! / (r!(n - r)!).
Chapter 5 - Binomial Theorem

Expansion of Binomials: General Term & Middle Term

Binomial Theorem

The Binomial Theorem provides a formula to expand binomials raised to a positive integer power. For any positive integer n, the expansion of (a + b)n is:

(a + b)n = ∑k=0n (nk) an-k bk

  • (nk) is the binomial coefficient, representing "n choose k".
  • an-k and bk are the respective powers of a and b in each term of the expansion.
  • The sum runs from k = 0 to k = n, resulting in n + 1 terms.

General Term in Binomial Expansion

The general term Tk in the binomial expansion of (a + b)n is given by:

Tk = (nk) an-k bk

  • Tk is the k-th term in the expansion (starting from k = 0).
  • (nk) is the binomial coefficient, which counts how many ways you can choose k items from n.
  • an-k and bk are the powers of a and b in the respective terms.

Example: Expansion of (a + b)4

Using the Binomial Theorem, the expansion of (a + b)4 is:

(a + b)4 = ∑k=04 (4k) a4-k bk

The terms are:

  • For k = 0: (40) a4 b0 = a4
  • For k = 1: (41) a3 b1 = 4a3 b
  • For k = 2: (42) a2 b2 = 6a2 b2
  • For k = 3: (43) a1 b3 = 4ab3
  • For k = 4: (44) a0 b4 = b4

Thus, the full expansion is:

a4 + 4a3 b + 6a2 b2 + 4ab3 + b4

Middle Term of Binomial Expansion

The middle term is the term that appears in the center of the expansion. Depending on whether n is odd or even, the middle term(s) differ:

  • For odd n, the middle term is the term corresponding to k = n/2.
  • For even n, there are two middle terms, corresponding to k = n/2 - 1 and k = n/2.

Example: Middle Term of (a + b)5

The general term is:

Tk = (5k) a5-k bk

The expansion is:

a5 + 5a4 b + 10a3 b2 + 10a2 b3 + 5ab4 + b5

Since n = 5 is odd, the middle term corresponds to k = 5/2 = 2.

The middle term is:

T2 = (52) a3 b2 = 10a3 b2

Example: Middle Terms of (a + b)6

The general term is:

Tk = (6k) a6-k bk

The expansion is:

a6 + 6a5 b + 15a4 b2 + 20a3 b3 + 15a2 b4 + 6ab5 + b6

Since n = 6 is even, there are two middle terms corresponding to k = 2 and k = 3:

  • For k = 2: T2 = (62) a4 b2 = 15a4 b2
  • For k = 3: T3 = (63) a3 b3 = 20a3 b3

Thus, the middle terms are 15a4 b2 and 20a3 b3.

Chapter 6 - Sequences & Series

1. Arithmetic Progression (AP)

What is an Arithmetic Progression (AP)?

An Arithmetic Progression (AP) is a sequence of numbers in which the difference between consecutive terms is constant. This constant difference is called the common difference.

General Form of an AP

The general form of an arithmetic progression is:

a, a + d, a + 2d, a + 3d, …

  • a is the first term,
  • d is the common difference,
  • n is the number of terms.

nth Term of an AP

The nth term of an arithmetic progression can be expressed by the formula:

Tn = a + (n - 1) ⋅ d

  • Tn is the nth term of the AP,
  • a is the first term,
  • d is the common difference,
  • n is the term number.

Sum of the First n Terms of an AP

The sum of the first n terms of an AP, denoted as Sn, is given by the formula:

Sn = (n / 2) ⋅ (2a + (n - 1) ⋅ d)

Alternatively, the sum can also be written as:

Sn = (n / 2) ⋅ (a + l)

  • n is the number of terms,
  • a is the first term,
  • d is the common difference,
  • l is the last term (the nth term).

Properties of an Arithmetic Progression

  • Constant difference: The difference between consecutive terms is constant, i.e., Tn+1 - Tn = d.
  • Midpoint of terms: The middle term (if the number of terms is odd) is the average of the first and last terms.
  • Sum of odd/even number of terms:
    • The sum of an odd number of terms is given by the product of the middle term and the number of terms.
    • For an even number of terms, the sum is the average of the first and last term multiplied by the number of terms.

Example of an Arithmetic Progression

Consider the AP: 3, 7, 11, 15, 19, …

The first term a = 3, and the common difference d = 4 (since 7 - 3 = 4).

Finding the nth Term:

To find the 6th term, use the formula:

Tn = a + (n - 1) ⋅ d

T6 = 3 + (6 - 1) ⋅ 4 = 3 + 5 ⋅ 4 = 3 + 20 = 23

So, the 6th term is 23.

Finding the Sum of the First 6 Terms:

We can use the formula:

Sn = (n / 2) ⋅ (2a + (n - 1) ⋅ d)

S6 = (6 / 2) ⋅ (2 ⋅ 3 + (6 - 1) ⋅ 4) = 3 ⋅ (6 + 20) = 3 ⋅ 26 = 78

So, the sum of the first 6 terms is 78.

2. Geometric Progression (GP)

What is a Geometric Progression (GP)?

A Geometric Progression (GP) is a sequence of numbers in which each term after the first is found by multiplying the previous term by a constant called the common ratio.

General Form of a GP

The general form of a geometric progression is:

a, ar, ar², ar³, ar⁴, …

  • a is the first term,
  • r is the common ratio (i.e., the factor by which each term is multiplied to get the next term),
  • n is the number of terms.

nth Term of a GP

The nth term of a geometric progression is given by the formula:

Tn = a ⋅ rn - 1

  • Tn is the nth term of the progression,
  • a is the first term,
  • r is the common ratio,
  • n is the term number.

Sum of the First n Terms of a GP

The sum of the first n terms of a geometric progression, denoted as Sn, is given by the formula:

Sn = a ⋅ (1 - rn) / (1 - r) (for r ≠ 1)

If the common ratio r = 1, the sum is simply:

Sn = n ⋅ a

Infinite Geometric Series

If the common ratio |r| < 1, the sum of an infinite geometric series (i.e., as n → ∞) is given by:

S∞ = a / (1 - r)

  • a is the first term,
  • r is the common ratio.

Properties of a Geometric Progression

  • Constant ratio: The ratio between consecutive terms is constant, i.e., Tn+1 / Tn = r.
  • Exponential growth/decay: If r > 1, the terms grow exponentially. If 0 < r < 1, the terms decrease exponentially.
  • Multiplicative nature: Each term is a product of the previous term and the common ratio.

Example of a Geometric Progression

Consider the GP: 2, 6, 18, 54, …

The first term a = 2, and the common ratio r = 3 (since 6 / 2 = 3).

Finding the nth Term:

To find the 5th term, use the formula:

Tn = a ⋅ rn - 1

T5 = 2 ⋅ 35 - 1 = 2 ⋅ 34 = 2 ⋅ 81 = 162

So, the 5th term is 162.

Finding the Sum of the First 4 Terms:

We can use the sum formula:

Sn = a ⋅ (1 - rn) / (1 - r)

S4 = 2 ⋅ (1 - 34) / (1 - 3)

S4 = 2 ⋅ (1 - 81) / -2 = 2 ⋅ (-80) / -2 = 80

So, the sum of the first 4 terms is 80.

3. AM-GM Inequality (Arithmetic Mean - Geometric Mean Inequality)

What is the AM-GM Inequality?

The AM-GM inequality is a fundamental inequality in mathematics that relates the Arithmetic Mean (AM) and Geometric Mean (GM) of a set of non-negative numbers. The inequality states that the Arithmetic Mean of a set of non-negative numbers is always greater than or equal to their Geometric Mean.

Statement of the AM-GM Inequality

For any set of n non-negative real numbers x₁, x₂, …, xₙ, the AM-GM inequality states:

x₁ + x₂ + ⋯ + xₙ / n ≥ x₁ ⋅ x₂ ⋅ ⋯ ⋅ xₙ1/n

  • x₁, x₂, …, xₙ are non-negative real numbers.
  • Arithmetic Mean (AM) is (x₁ + x₂ + ⋯ + xₙ) / n.
  • Geometric Mean (GM) is (x₁ ⋅ x₂ ⋅ ⋯ ⋅ xₙ)1/n.

Equality Condition

Equality holds if and only if x₁ = x₂ = ⋯ = xₙ. That is, the arithmetic mean equals the geometric mean only when all the numbers are equal.

Example of AM-GM Inequality

Let's consider the numbers 4 and 16.

Arithmetic Mean (AM):

AM = (4 + 16) / 2 = 10

Geometric Mean (GM):

GM = √(4 × 16) = √64 = 8

According to the AM-GM inequality:

AM = 10 ≥ GM = 8

So, the AM is greater than the GM, which follows the AM-GM inequality.

Equality Condition in Action

For equality to hold, all the numbers must be the same. For example, if we take the numbers 5 and 5, then both the arithmetic and geometric means will be equal:

Arithmetic Mean (AM):

AM = (5 + 5) / 2 = 5

Geometric Mean (GM):

GM = √(5 × 5) = √25 = 5

In this case, AM = GM = 5.

Applications of the AM-GM Inequality

  • Optimization Problems: The AM-GM inequality is widely used to solve optimization problems, especially those involving product and sum constraints.
  • Inequalities: It is used to prove many other inequalities in mathematics, such as the Cauchy-Schwarz inequality and Holder's inequality.
  • Economics: In economics, the AM-GM inequality is applied in the analysis of production and consumption.

Generalized AM-GM Inequality

For n non-negative numbers x₁, x₂, …, xₙ, the AM-GM inequality can be generalized as follows:

p₁x₁ + p₂x₂ + ⋯ + pₙxₙ / (p₁ + p₂ + ⋯ + pₙ) ≥ (x₁p₁ ⋅ x₂p₂ ⋅ ⋯ ⋅ xₙpₙ)1/n

Where p₁, p₂, …, pₙ are positive real numbers.

3. Differentiability in Calculus

Definition of Differentiability

  • A function \( f(x) \) is differentiable at \( x = a \) if the derivative exists at that point. Mathematically, the derivative of \( f(x) \) at \( x = a \) is:
    • f ' ( a ) = lim h → 0 f(a+h) − f(a) h

Differentiability vs Continuity

  • A function must be continuous at \( x = a \) to be differentiable there, but a continuous function may not be differentiable.
  • Differentiability implies continuity, but the reverse is not always true. For example, a function can be continuous at a point but not differentiable there (like \( f(x) = |x| \) at \( x = 0 \)).

Conditions for Differentiability

  • Continuity at the point: The function must be continuous at \( x = a \).
  • No sharp corners: The function must not have any sharp corners or cusps at \( x = a \).
  • No vertical tangents: The function must not have a vertical tangent at \( x = a \).

Differentiability of Common Functions

  • Polynomials: All polynomial functions are differentiable everywhere.

    • Example: f(x) = x 3 − 3 x + 2

  • Rational Functions: Differentiable where the denominator is non-zero.

    • Example: f(x) = 1 x for x ≠ 0

  • Trigonometric Functions: Basic trigonometric functions like \( \sin(x) \), \( \cos(x) \), and \( \tan(x) \) are differentiable wherever they are defined.

    • Example: f(x) = sin(x)

Examples of Differentiability

  • Example 1: \( f(x) = x^2 \) is differentiable everywhere, and its derivative is \( f'(x) = 2x \).
  • Example 2: \( f(x) = |x| \) is not differentiable at \( x = 0 \) due to a sharp corner.
  • Example 3: \( f(x) = \frac{1}{x} \) is differentiable for \( x ≠ 0 \), but not at \( x = 0 \).

1. Indefinite & Definite Integrals

Indefinite Integral (Antiderivative)

  • The indefinite integral represents a family of functions whose derivative is the given function. It is also called the antiderivative because it essentially "reverses" differentiation.

Definition

  • ∫f(x) dx = F(x) + C

    • Where:

    • f(x) is the integrand (the function to be integrated).
    • F(x) is the antiderivative of f(x), i.e., F'(x) = f(x).
    • C is the constant of integration, representing the fact that there are infinitely many antiderivatives.

Basic Rules

  • Power Rule: ∫xⁿ dx = (xⁿ⁺¹) / (n+1) + C (for n ≠ -1)
  • Constant Rule: ∫c dx = c * x + C
  • Sum/Difference Rule: ∫(f(x) ± g(x)) dx = ∫f(x) dx ± ∫g(x) dx
  • Exponential Rule: ∫eˣ dx = eˣ + C
  • Trigonometric Functions:
    • ∫sin(x) dx = -cos(x) + C
    • ∫cos(x) dx = sin(x) + C

Definite Integral

The definite integral computes the signed area under the curve of a function between two specific limits, a and b. Unlike the indefinite integral, the definite integral results in a numerical value rather than a function.

Definition

  • ∫ab f(x) dx = F(b) - F(a)

    • Where:

    • f(x) is the integrand.
    • F(x) is an antiderivative of f(x).
    • a and b are the limits of integration.

Interpretation

The definite integral represents the net area between the curve f(x) and the x-axis over the interval [a, b]. If the curve lies above the axis, the area is positive; if it lies below, the area is negative.

Properties

  • Additivity: ∫ab f(x) dx + ∫bc f(x) dx = ∫ac f(x) dx
  • Reversal of Limits: ∫ab f(x) dx = -∫ba f(x) dx
  • Constant Factor: ∫ab c ⋅ f(x) dx = c ⋅ ∫ab f(x) dx
  • Zero Integral: If f(x) = 0, then ∫ab 0 dx = 0

Fundamental Theorem of Calculus

The Fundamental Theorem of Calculus connects the concepts of differentiation and integration. It has two parts:

  • Part 1: If f(x) is continuous on [a, b] and F(x) is an antiderivative of f(x), then: ∫ab f(x) dx = F(b) - F(a)
  • Part 2: If F(x) is defined as the integral of f(x) from a to x, then F'(x) = f(x). This shows that differentiation and integration are inverse operations.

Examples

Indefinite Integral:

∫3x² dx = x³ + C

Here, f(x) = 3x², and the antiderivative is x³, with the constant of integration C.

Definite Integral:

∫13 x² dx

First, find the antiderivative of x², which is x³/3. Then, evaluate the definite integral:

[x³/3]13 = (27/3) - (1/3) = 26/3

2. Properties and Applications of Definite Integrals: Areas Under Curves

Properties of Definite Integrals

Additivity Property

If a function f(x) is continuous on an interval [a, c], then the definite integral over this interval can be split into the sum of integrals over subintervals:

∫ac f(x) dx = ∫ab f(x) dx + ∫bc f(x) dx

This allows you to break down a large integral into smaller, more manageable parts.

Reversal of Limits

Reversing the limits of integration changes the sign of the integral:

∫ab f(x) dx = -∫ba f(x) dx

This property ensures consistency when switching the order of integration.

Linearity

If f(x) and g(x) are functions and c is a constant, then:

∫ab [c f(x) + g(x)] dx = c ∫ab f(x) dx + ∫ab g(x) dx

This property allows you to separate terms in an integral and factor out constants.

Zero Integral

If the function f(x) = 0 over the interval [a, b], the integral is zero:

∫ab 0 dx = 0

This is because there is no area under the curve of the zero function.

Non-negativity

If f(x) is non-negative on the interval [a, b], then the integral will also be non-negative:

f(x) ≥ 0 ⇒ ∫ab f(x) dx ≥ 0

This is because the area under the curve is always positive or zero.

Fundamental Theorem of Calculus

If f(x) is continuous on [a, b] and F(x) is an antiderivative of f(x), then: 

∫ab f(x) dx = F(b) - F(a)

This links the concept of differentiation and integration.

Applications of Definite Integrals: Areas Under Curves

1. Area Between a Curve and the X-Axis

To find the area between a curve y = f(x) and the x-axis from x = a to x = b, we compute the definite integral:

Area = ∫ab f(x) dx

This represents the net area under the curve. If f(x) is above the x-axis, the area is positive; if f(x) is below the x-axis, the area is negative.

Example:

Find the area under the curve f(x) = x² from x = 1 to x = 3.

∫13 x² dx = [x³ / 3]13 = (27 / 3) - (1 / 3) = 26 / 3

Thus, the area under the curve from x = 1 to x = 3 is 26 / 3.

2. Area Between Two Curves

To find the area between two curves, say y = f(x) and y = g(x), the formula is:

Area = ∫ab [f(x) - g(x)] dx

This gives the area between the two curves from x = a to x = b, where f(x) ≥ g(x).

Example:

Find the area between the curves y = x² and y = x from x = 0 to x = 1.

Area = ∫01 [x - x²] dx = [(x² / 2) - (x³ / 3)]01 = (1 / 2 - 1 / 3) - (0) = 1 / 6

Thus, the area between the curves from x = 0 to x = 1 is 1 / 6.

If the curve is expressed in terms of y = f(x), and we want to find the area with respect to the y-axis, the integral is set up differently:

Area = ∫y₁y₂ f-1(y) dy

Where f-1(y) is the inverse function.

The average value of a function f(x) over the interval [a, b] is given by: 

Average value of f(x) = (1 / (b - a)) ∫ab f(x) dx

This gives the mean height of the function over the interval.

1. Order and Degree of a Polynomial

Definition

  • The **order** of a polynomial refers to the highest power of the variable \(x\) in the polynomial.
  • The **degree** of a polynomial is the highest exponent of the variable \(x\) in the polynomial, indicating the highest power of the variable that appears with a non-zero coefficient.

Order of a Polynomial

  • The order of a polynomial is simply the highest power of the variable \(x\) in the polynomial.
  • The order tells you how many times the variable is multiplied by itself in the term with the highest exponent.

Example

  • For the polynomial \(P(x) = 4x^3 + 2x^2 - 7x + 5\), the order is 3 because the highest power of \(x\) is \(x^3\).

Degree of a Polynomial

  • The degree of a polynomial is the highest exponent of the variable \(x\) when the polynomial is written in standard form (terms ordered from highest to lowest powers).
  • The degree indicates the highest power of the variable with a non-zero coefficient.

Example

  • For the polynomial \(P(x) = 4x^3 + 2x^2 - 7x + 5\), the degree is 3 because the highest degree term is \(4x^3\), where the exponent of \(x\) is 3.

Difference Between Order and Degree

  • The **degree** refers to the highest exponent of \(x\), while the **order** is often used interchangeably with degree, particularly in polynomials.
  • The degree applies to both monomials (single terms) and polynomials (sum of terms).

Additional Notes

  • For a constant polynomial (e.g., \(P(x) = 5\)), the degree is 0 because there is no variable.
  • If a polynomial has multiple terms with the highest exponent (e.g., \(P(x) = 3x^2 + 2x^2 + 5\)), the degree remains the highest exponent of \(x\), which in this case is 2.

Examples

  1. Example 1: Polynomial with Degree 5

    • Given the polynomial \(P(x) = 2x^5 - 3x^4 + x^2 - 7\), the degree is 5 because the highest power of \(x\) is \(x^5\). The order is also 5.

  2. Example 2: Polynomial with Degree 1 (Linear Polynomial)

    • Given the polynomial \(P(x) = 3x - 4\), the degree is 1 because the highest power of \(x\) is \(x^1\). The order is also 1.

2. Formation of Differential Equations

Definition

  • A differential equation is an equation that involves derivatives of a function or a set of functions.
  • They can be used to model a wide range of phenomena, such as population growth, heat conduction, and mechanical vibrations.

General Method to Form a Differential Equation

  • Start with a given equation that involves variables and their derivatives.
  • Eliminate arbitrary constants to get an equation involving the function and its derivatives.
  • If there is an arbitrary constant, differentiate the equation with respect to the independent variable (usually x or t) to get rid of the constant and introduce a derivative.
  • Write the equation in terms of derivatives of the function. The final equation will involve the dependent variable and its derivatives.

Example 1: Forming a Differential Equation from a General Function

Suppose we are given the equation:

y = a x 2 + b x + c

where a, b, and c are constants.

Step 1: Differentiate the equation to eliminate constants

  • Differentiate with respect to x:
    • d y / d x = 2 a x + b
  • Then, differentiate again:
    • d 2 y / d x 2 = 2 a

Step 2: Eliminate the constants

  • From the equation d 2 y / d x 2 = 2 a , we can see that the second derivative of y is a constant, 2a. Thus, the differential equation becomes:
    • d 2 y / d x 2 = 2 a

This is a second-order linear differential equation.

Example 2: Forming a Differential Equation from a Given Relationship

Given the relationship between x and y:

y = e k x

where k is a constant.

Step 1: Differentiate the equation

  • Differentiate with respect to x:
    • d y / d x = k e kx

Step 2: Eliminate the constant k

  • Since y = e kx , we can solve for k in terms of x and y:
    • k = 1 x ln ( y )
  • Substitute this back into the first derivative equation to eliminate the constant k.

After simplification, the final differential equation would involve x, y, and derivatives of y.

Example 3: Forming a Differential Equation from a Physical Situation

Suppose a population grows according to the equation:

d P / d t = k P

where P is the population and k is a constant of proportionality. This is a first-order linear differential equation representing exponential growth.

Step 1: Write the equation

The rate of change of the population P with respect to time t is proportional to the population size itself:

d P / d t = k P

This equation directly represents the rate of change of P, so it does not need further differentiation or simplification. It is already a differential equation.

General Form of a First-Order Differential Equation

A general first-order differential equation can be written as:

F ( x , y , d y / d x ) = 0

Where F is some function of x, y, and dy/dx.

Example 4: Formation from Initial Conditions

Given the relationship:

y = ln ( x ) + C

Where C is a constant, we can form a differential equation by eliminating C.

Step 1: Differentiate the equation

d y / d x = 1 x

Step 2: Eliminate the constant C

Since there is no explicit constant term in the equation after differentiation, the differential equation formed is:

d y / d x = 1 x

1. Straight Lines - Detailed Notes

Equation of a Straight Line

A straight line in the coordinate plane can be represented by an equation. There are various forms depending on the given information.

a. General Form of the Equation

The general form of the equation of a straight line is:

A x + B y + C = 0
  • Where A, B, and C are constants. This is the most general form and can represent any straight line.
  • The slope of the line is m = -A B (if B ≠ 0).
  • The x-intercept and y-intercept can be derived from this equation.

b. Slope-Intercept Form

The slope-intercept form is one of the most commonly used forms to express a straight line. It is given by:

y = m x + b
  • Where m is the slope of the line (the rate of change of y with respect to x).
  • b is the y-intercept, which is the point where the line intersects the y-axis (i.e., x = 0).

Example: For the line equation y = 2 x + 3 :

  • The slope m = 2, meaning for every unit increase in x, y increases by 2 units.
  • The y-intercept b = 3, meaning the line crosses the y-axis at (0, 3).

c. Point-Slope Form

The point-slope form of the equation of a line is useful when we know the slope of the line and a specific point on the line. It is given by:

y - y1 = m ( x - x1 )
  • (x1, y1) is a point on the line.
  • m is the slope of the line.

Example: For a line passing through the point (1, 2) with a slope of 3:

y - 2 = 3 ( x - 1 )

This simplifies to y = 3 x - 1 .

d. Intercept Form

The intercept form is particularly useful when we know the intercepts of the line on both the x-axis and the y-axis. It is given by:

x a y b c

2. Circle

Definition

  • A circle is a geometric shape defined as the set of all points in a plane that are equidistant from a fixed point called the center.
  • The distance from the center to any point on the circle is called the radius.

Equation of a Circle

  • The standard form of the equation of a circle with center \((h, k)\) and radius \(r\) is: x - h 2 + y - k 2 = r 2
  • For example, for a circle with center \((3, -2)\) and radius 5, the equation becomes: x - 3 2 + y + 2 2 = 5 2
  • The general form of the equation of a circle is: x 2 + y 2 + 2 g x + 2 f y + c = 0
  • From the general form, we can find the center and radius:
    • Center: \((-g, -f)\)
    • Radius: \(r = \sqrt{g^2 + f^2 - c}\)

Key Properties of a Circle

  • Center: The point equidistant from all points on the circle.
  • Radius: The constant distance from the center to any point on the circle.
  • Diameter: The longest chord of the circle, which is twice the radius. D = 2 r
  • Chord: A line segment joining any two points on the circle. The diameter is the longest chord.
  • Arc: A part of the circumference of a circle. A minor arc is the shorter segment, while the major arc is the longer one.
  • Circumference: The total length around the circle. The formula is: C = 2 π r
  • Area: The area enclosed by the circle is given by: A = π r 2

Tangent to a Circle

  • A tangent is a line that touches the circle at exactly one point, called the point of tangency.
  • The tangent is always perpendicular to the radius at the point of tangency.
  • For a tangent to a circle with center \((h, k)\) at a point \((x_1, y_1)\) on the circle, the equation is: x - h 1 ( x - h ) + y - k 1 ( y - k ) = r 2

Power of a Point

  • The power of a point \(P\) with respect to a circle is the square of the distance from the point to the center minus the square of the radius. If the point lies outside the circle and a secant from \(P\) intersects the circle at points \(A\) and \(B\), the power of point \(P\) is given by: Power = PA × PB

Applications of Circles

  • Geometry: Circles are used to model things like wheels, gears, or orbits in physics.
  • Trigonometry: The unit circle plays a key role in trigonometric functions and identities.
  • Astronomy: Circles are used to describe orbits of planets and other celestial bodies.

3. Ellipse

Definition

  • An ellipse is a geometric shape formed by the set of all points in a plane where the sum of the distances from two fixed points, called foci, is constant.
  • It can be thought of as a stretched circle.

Equation of an Ellipse

  • The standard form of the equation of an ellipse with center at \((h, k)\), horizontal radius \(a\), and vertical radius \(b\) is: (x - h) 2 a 2 + (y - k) 2 b 2 = 1
  • For a circle, where \(a = b\), the equation simplifies to the equation of a circle.
  • Example: For an ellipse with center at \((2, 3)\), horizontal radius 5, and vertical radius 3, the equation is: (x - 2) 2 5 2 + (y - 3) 2 3 2 = 1

Key Properties of an Ellipse

  • Foci: An ellipse has two foci. The sum of the distances from any point on the ellipse to the two foci is constant.
  • Major and Minor Axes: The longest diameter of the ellipse is called the major axis, and the shortest diameter is the minor axis. The center is the midpoint of both axes.
  • Vertices: The points where the ellipse intersects the major axis are called the vertices.
  • Co-vertices: The points where the ellipse intersects the minor axis are called the co-vertices.

Relations Between Axes and Foci

  • The distance between the center and each focus is \(c\), where: c = a 2 - b 2 1
  • If \(a = b\), the ellipse becomes a circle, and the foci coincide with the center.

Area of an Ellipse

  • The area of an ellipse is given by the formula: A = π a b

Applications of Ellipses

  • Astronomy: The orbits of planets and celestial bodies are elliptical.
  • Optics: Elliptical mirrors and lenses focus light effectively at the foci.
  • Engineering: Ellipses are used in the design of gears, wheels, and mechanical systems.

4. Parabola

Definition

  • A parabola is a U-shaped curve that can open upward, downward, leftward, or rightward. It is a type of conic section that can be defined geometrically as the set of all points equidistant from a fixed point called the focus and a fixed line called the directrix. Parabolas are used in a variety of fields, from physics (projectile motion) to engineering (reflecting surfaces).

1. General Equation of a Parabola

The equation of a parabola depends on its orientation in the coordinate plane and the location of its vertex.

a. Standard Equation of a Parabola with a Vertical Axis (Up or Down Opening):

  • The equation for a parabola that opens upward or downward is: y = a ( x - h ) 2 + k
  • Where:
    • \((h, k)\) is the vertex of the parabola.
    • \(a\) determines the width and direction of the parabola:
      • If \(a > 0\), the parabola opens upward.
      • If \(a < 0\), the parabola opens downward.
  • Example: The equation y = 2 ( x - 1 ) 2 + 3 represents a parabola that opens upward, with the vertex at \((1, 3)\).

b. Standard Equation of a Parabola with a Horizontal Axis (Left or Right Opening):

  • The equation for a parabola that opens leftward or rightward is: x = a ( y - k ) 2 + h
  • Where:
    • \((h, k)\) is the vertex of the parabola.
    • \(a\) determines the width and direction of the parabola:
      • If \(a > 0\), the parabola opens rightward.
      • If \(a < 0\), the parabola opens leftward.
  • Example: The equation x = - 3 ( y + 2 ) 2 + 4 represents a parabola that opens leftward, with the vertex at \((4, -2)\).

2. Properties of a Parabola

  • Vertex: The vertex of a parabola is the point where the curve changes direction. It is the minimum point for a parabola that opens upward and the maximum point for a parabola that opens downward (for vertical parabolas). The vertex in the standard form \(y = a(x - h)^2 + k\) is \((h, k)\).
  • Focus: The focus of a parabola is a fixed point located on the axis of symmetry, inside the curve. It is equidistant from any point on the parabola. For vertical parabolas, the focus is at: ( h , k + 1 4a ) if the parabola opens upward, or: ( h , k - 1 4a ) if it opens downward.
  • Directrix: The directrix is a line that is equidistant from any point on the parabola. It lies opposite to the focus and helps define the curve. For vertical parabolas, the directrix is at: y = k - 1 4a if the parabola opens upward, or: y = k + 1 4a if it opens downward.
  • Axis of Symmetry: The axis of symmetry is the line that passes through the vertex and the focus, dividing the parabola into two mirror-image halves. For a vertical parabola \(y = a(x - h)^2 + k\), the axis of symmetry is the vertical line \(x = h\). For a horizontal parabola \(x = a(y - k)^2 + h\), the axis of symmetry is the horizontal line \(y = k\).

3. Equation of a Parabola in General Form

  • The general form of the equation of a parabola is: A x 2 + B x y + C y 2 + D x + E y + F = 0
  • Where \(A\), \(B\), \(C\), \(D\), \(E\), and \(F\) are constants. This form can be converted into the standard form by completing the square, and it helps to determine the orientation and shape of the parabola.

4. Focal Length

  • The focal length of a parabola is the distance from the vertex to the focus. It is related to the coefficient \(a\) in the equation:
    • For a parabola with a vertical axis of symmetry, the focal length is \(\frac{1}{4a}\).
    • For a parabola with a horizontal axis of symmetry, the focal length is also \(\frac{1}{4a}\).

5. Tangent to a Parabola

  • The equation of the tangent to the parabola at any point \((x_1, y_1)\) on the curve can be written as: y - y 1 = m ( x - x 1 )
  • Where \(m\) is the slope of the tangent. For vertical parabolas, the slope can be derived using the derivative of the equation of the parabola, and similarly, for horizontal parabolas, the slope can be determined.

6. Parametric Equations of a Parabola

  • For a parabola with a vertical axis, the parametric equations are:
    • x = a t 2 + h
    • y = 2 a t + k
  • For a parabola with a horizontal axis, the parametric equations are:
    • x = 2 a t + h
    • y = a t 2 + k

7. Applications of Parabolas

  • Projectile Motion: The path of an object thrown in the air follows a parabolic trajectory. The equation of motion for an object under gravity is often modeled by a parabola.
  • Satellite Dishes: Parabolic reflectors are used in satellite dishes because they reflect incoming signals to a single point, the focus, where the receiver is located.
  • Car Headlights: Parabolic reflectors are used in car headlights, focusing light into a beam that is directed in a specific direction.
  • Engineering: Parabolic arches and bridges utilize the structural properties of parabolas to support heavy loads with minimal material.

8. Example of a Parabola

Consider the equation of a parabola: y = 3 ( x - 2 ) 2 + 4

  • The vertex is at \((2, 4)\).
  • Since \(a = 3 > 0\), the parabola opens upward.
  • The focus is located at \((2, 4 + \frac{1}{4 \times 3}) = (2, 4 + \frac{1}{12}) = (2, \frac{49}{12})\).
  • The directrix is the line \(y = 4 - \frac{1}{12} = \frac{47}{12}\).

4. Intersection

Intersection of Two Planes

  • The intersection of two planes generally results in a line.
  • For two planes with equations: A 1 x + B 1 y + C 1 z + D 1 = 0 A 2 x + B 2 y + C 2 z + D 2 = 0
  • The direction ratios of the line of intersection can be found using the cross product of the normal vectors of the planes:
    • Normal vector of the first plane: n ₁ = < A1 , B1 , C1 >
    • Normal vector of the second plane: n ₂ = < A2 , B2 , C2 >
    • Direction vector of the line of intersection: L = n ₁ × n ₂
  • To find the equation of the line, a point on the line must be determined by solving the system of equations for x, y, and z.

Intersection of Three Planes

  • The intersection of three planes can result in:
    • A single point if the planes are not parallel and their normal vectors are not proportional.
    • A line if two planes are parallel and the third intersects them along a line.
    • No intersection if the planes are parallel or form a skew configuration.
  • To find the point of intersection of three planes, solve the system of three equations:
    • A 1 x + B 1 y + C 1 z + D 1 = 0
    • A 2 x + B 2 y + C 2 z + D 2 = 0
    • A 3 x + B 3 y + C 3 z + D 3 = 0
  • The solution to this system gives the coordinates (x, y, z) of the point of intersection.

Parallel Planes

  • If two planes are parallel, they will not intersect. The normal vectors of the planes are proportional:
    • n ₁ ∥ n ₂
  • The planes will either be distinct or coincident if the constant terms are the same. Example of two parallel planes:
    • A 1 x + B 1 y + C 1 z + D 1 = 0
    • A 2 x + B 2 y + C 2 z + D 2 = 0

Coincident Planes

  • If two planes are coincident, they are the same plane. The equations of the two planes are either identical or proportional:
    • A 1 x + B 1 y + C 1 z + D 1 = 0
    • k ( A 1 x + B 1 y + C 1 z + D 1 ) = 0

Parallel and Non-Intersecting Planes

  • If the planes are neither parallel nor coincident, they will intersect along a straight line. To find the line of intersection:
    1. Write the equations of the two planes:
      • A 1 x + B 1 y + C 1 z + D 1 = 0
      • A 2 x + B 2 y + C 2 z + D 2 = 0
    2. Find the direction ratios of the line by taking the cross product of the normal vectors of the planes.
    3. Find a point on the line by solving the system of two plane equations for two variables, then substitute to find the third variable.
    4. Write the parametric equation of the line:
      • x = x0 + t Lx
      • y = y0 + t Ly
      • z = z0 + t Lz

Definition

Vectors:

  • Acceleration (m/s²)

2. Components of Vectors

Definition

  • A vector can be broken down into components along specific directions, usually along the coordinate axes of a system (typically the x-axis, y-axis, and z-axis in 3D space).
  • The process of breaking a vector into components simplifies vector operations and allows for easier analysis in different directions.

Components of a Vector in 2D

  • In a two-dimensional Cartesian coordinate system, any vector A can be written as a sum of its components along the x-axis and y-axis.
  • Consider a vector A having magnitude A and direction θ (angle with the positive x-axis).
  • The vector A can be represented as:

    A = Ax i + Ay j

    • Ax is the x-component of the vector A
    • Ay is the y-component of the vector A
    • i and j are unit vectors in the direction of the x-axis and y-axis, respectively.
  • Using trigonometry, the components of vector A are given by:
    • Ax = A * cos(θ) (the x-component)
    • Ay = A * sin(θ) (the y-component)
  • Thus, the vector A in component form becomes:

    A = Ax i + Ay j = A * cos(θ) i + A * sin(θ) j

Example (2D)

  • If A has a magnitude of 5 units and makes an angle of 30° with the positive x-axis, then:
    • Ax = 5 * cos(30°) = 5 * √3/2 = 4.33
    • Ay = 5 * sin(30°) = 5 * 1/2 = 2.5
  • So, the vector A in component form is:

    A = 4.33 i + 2.5 j

Components of a Vector in 3D

  • In a three-dimensional Cartesian coordinate system, any vector A can be written as a sum of its components along the x-axis, y-axis, and z-axis.
  • Consider a vector A in 3D space, which can be represented as:

    A = Ax i + Ay j + Az k

    • Ax is the x-component of the vector A
    • Ay is the y-component of the vector A
    • Az is the z-component of the vector A
    • i, j, k are unit vectors along the x-axis, y-axis, and z-axis, respectively.
  • If you know the magnitude A of the vector and the angles θx, θy, and θz that the vector makes with the coordinate axes, you can find the components as follows:
    • Ax = A * cos(θx)
    • Ay = A * cos(θy)
    • Az = A * cos(θz)
  • Alternatively, if the direction cosines are given, the components can be directly calculated using:
    • Ax = A * cos(α)
    • Ay = A * cos(β)
    • Az = A * cos(γ)

Example (3D)

  • Suppose a vector A has a magnitude of 6 units and the direction angles are:
    • α = 45° with the x-axis
    • β = 60° with the y-axis
    • γ = 90° with the z-axis
  • The components of the vector are:
    • Ax = 6 * cos(45°) = 6 * √2/2 = 4.24
    • Ay = 6 * cos(60°) = 6 * 1/2 = 3
    • Az = 6 * cos(90°) = 6 * 0 = 0
  • Thus, the vector A in component form is:

3. Dot Product(Scalar Product)

The dot product (also called the scalar product) is a mathematical operation that takes two vectors and returns a scalar (a number, not a vector). It is a measure of how much one vector extends in the direction of another vector.

Definition:

For two vectors A = Ax i + Ay j + Az k and B = Bx i + By j + Bz k, their dot product A ⋅ B is defined as:

A ⋅ B = Ax Bx + Ay By + Az Bz

where:

  • Ax, Ay, Az are the components of A
  • Bx, By, Bz are the components of B

Alternatively, the dot product can be written in terms of the magnitudes of the vectors and the cosine of the angle θ between them:

A ⋅ B = |A| |B| cos(θ)

where:

  • |A| is the magnitude (or length) of vector A
  • |B| is the magnitude of vector B
  • θ is the angle between the two vectors.

Geometric Interpretation:

The dot product of two vectors gives a measure of how much one vector "projects" onto the other. If the angle between the vectors is 0° (i.e., the vectors are parallel), the dot product is at its maximum, equal to the product of the magnitudes of the vectors. If the vectors are perpendicular (i.e., the angle between them is 90°), the dot product is zero, indicating that there is no projection of one vector along the other.

Properties of the Dot Product:

  • Commutative Property: The dot product is commutative, meaning:
    A ⋅ B = B ⋅ A
  • Distributive Property: The dot product is distributive over vector addition:
    A ⋅ (B + C) = A ⋅ B + A ⋅ C
  • Scalar Multiplication: The dot product of a scalar multiplied by a vector with another vector is:
    (k A) ⋅ B = k(A ⋅ B), where k is a scalar.
  • Result is a Scalar: The result of the dot product is always a scalar (real number), not a vector.
  • Zero Dot Product: If A ⋅ B = 0, then the vectors A and B are perpendicular (orthogonal), i.e., the angle between them is 90°.

Let A = 3i + 2j + 4k and B = 1i + 0j + 5k. The dot product A ⋅ B is calculated as:

2. Median

Definition

  • The median is a measure of central tendency, which represents the middle value of a data set when it is ordered in ascending or descending order.

Steps to Calculate the Median

  • Arrange the data: Arrange the data set in ascending or descending order.
  • Find the middle value:
    • For an odd number of data points: The median is the middle value.
    • For an even number of data points: The median is the average of the two middle values.

Formula

    Where n is the total number of values in the data set.

    Where n is the total number of values in the data set, and xn/2 and xn/2+1 are the middle two values.

3. Mode

Definition

  • If one value appears most frequently, it is unimodal.
  • If two values appear with the same highest frequency, the data set is bimodal.

Properties of the Mode

  • Can Have More Than One Mode: A data set can be unimodal, bimodal, or multimodal, depending on the frequency of the values.

4. Variance

Definition

  • Variance is a statistical measure that represents the spread or dispersion of a set of values. It quantifies how far each number in the data set is from the mean (average) and, therefore, from every other number in the data set. A higher variance indicates that the numbers are more spread out, while a lower variance suggests that the numbers are closer to the mean.

Formula

  • For a population:
    • σ 2 = ∑ xi - μ n

    Where σ² is the population variance, xᵢ are the data points, μ is the population mean, and n is the number of data points.

  • For a sample:
    • s 2 = ∑ xi - x̅ n-1

    Where s² is the sample variance, xᵢ are the data points, x̅ is the sample mean, and n is the number of data points in the sample.

Steps to Calculate Variance

  • Find the Mean: First, calculate the mean (average) of the data set.
  • Subtract the Mean: Subtract the mean from each data point.
  • Square the Differences: Square each of the differences obtained in step 2.
  • Find the Average of Squared Differences:
    • For population variance, divide the sum of squared differences by n.
    • For sample variance, divide the sum of squared differences by n-1.

Properties of Variance

  • Non-negative: Variance is always zero or positive because it involves squaring differences (negative values become positive).
  • Units: The units of variance are the square of the units of the data. For example, if the data is in meters, the variance will be in square meters.
  • Sensitive to Outliers: Variance is sensitive to extreme values (outliers) because large differences from the mean are squared, leading to a larger variance.

Example

  • For the data set {3, 5, 7}:
    • Mean: μ = 3 + 5 + 7 3 = 5
    • Differences from the mean: (3 - 5) = -2, (5 - 5) = 0, (7 - 5) = 2
    • Squared differences: (-2)² = 4, 0² = 0, 2² = 4
    • Sum of squared differences: 4 + 0 + 4 = 8
    • Variance: σ² = 8/3 = 2.67 (for population variance)
  • For a sample (with 3 data points), the sample variance would be:
    • Variance: s² = 8/(3-1) = 8/2 = 4

Applications of Variance

  • In Finance: Variance is used to measure the risk or volatility of an investment's returns.
  • In Quality Control: It helps in assessing the consistency or variation in manufacturing processes.
  • In Data Analysis: It is used to understand the spread of data and the reliability of the mean.

5. Standard Deviation

Definition

  • Standard Deviation is a measure of the amount of variation or dispersion of a set of values. It is the square root of the variance and is used to describe how much individual data points deviate from the mean. Unlike variance, which is in squared units, the standard deviation is in the same units as the original data, making it easier to interpret.

Formula

  • Population Standard Deviation:
    • σ = ∑ xi - μ n

    Where σ is the population standard deviation, xᵢ are the data points, μ is the population mean, and n is the number of data points in the population.

  • Sample Standard Deviation:
    • s = ∑ xi - x̅ n-1

    Where s is the sample standard deviation, xᵢ are the data points, x̅ is the sample mean, and n is the number of data points in the sample.

Steps to Calculate Standard Deviation

  • Find the Mean: Calculate the mean of the data set.
  • Subtract the Mean: Subtract the mean from each data point to find the deviation from the mean.
  • Square the Differences: Square each of the deviations.
  • Find the Average of Squared Differences:
    • For population standard deviation, divide the sum of squared differences by n.
    • For sample standard deviation, divide the sum of squared differences by n-1.
  • Square Root: Take the square root of the variance to get the standard deviation.

Properties of Standard Deviation

  • Non-negative: The standard deviation is always non-negative because it involves taking the square root of a squared quantity.
  • Same Units as Data: Unlike variance, which has squared units, the standard deviation is in the same units as the original data, making it easier to interpret.
  • Sensitive to Outliers: Like variance, the standard deviation is sensitive to extreme values, or outliers, because it involves squaring deviations.
  • Informs on Spread: A larger standard deviation means the data points are spread out more widely from the mean, while a smaller standard deviation indicates the data points are closer to the mean.

Example

  • For the data set {2, 4, 6, 8}:
    • Mean: μ = 2 + 4 + 6 + 8 4 = 5
    • Deviations from the mean: (2 - 5) = -3, (4 - 5) = -1, (6 - 5) = 1, (8 - 5) = 3
    • Squared deviations: (-3)² = 9, (-1)² = 1, (1)² = 1, (3)² = 9
    • Sum of squared deviations: 9 + 1 + 1 + 9 = 20
    • Variance (Population): σ² = 20/4 = 5
    • Standard Deviation (Population): σ = √5 ≈ 2.24
  • For a sample (with 4 data points), the sample standard deviation would be:
    • Variance: s² = 20/(4-1) = 20/3 ≈ 6.67
    • Standard Deviation (Sample): s = √6.67 ≈ 2.58

Applications of Standard Deviation

  • In Finance: To measure the volatility or risk of an investment.
  • In Quality Control: To assess the consistency or precision of manufacturing processes.
  • In Education: To measure how spread out students' scores are from the average score.

1. Basics of Probability

Definition

  • Probability is a branch of mathematics that deals with the likelihood or chance of an event happening. It is used to predict the outcome of uncertain events and is a fundamental concept in statistics, mathematics, and various fields like finance, engineering, and science.

Basic Terms in Probability

  • Experiment: A process or action that results in an outcome. Example: Tossing a coin, rolling a die.
  • Sample Space (S): The set of all possible outcomes of an experiment. Example: For a coin toss, the sample space is S = {H, T}, where H stands for heads and T for tails.
  • Event: A specific outcome or a set of outcomes from the sample space. Example: "Getting a head" is an event in the coin toss experiment.
  • Outcome: A specific result of an experiment. Example: "Heads" is an outcome in a coin toss.
  • Favorable Outcomes: The outcomes in the sample space that satisfy the event.

Probability of an Event

  • The probability of an event is a number between 0 and 1 that indicates how likely the event is to occur.
    • P ( A ) = Number of favorable outcomes Total number of outcomes in sample space

    Where:

    • P(A) is the probability of event A.
    • The value of P(A) ranges from 0 to 1:
      • P(A) = 0: The event will not occur.
      • P(A) = 1: The event is certain to occur.
      • 0 < P(A) < 1: The event has a certain likelihood of occurring.

Types of Events

  • Simple Event: An event with a single outcome. Example: Rolling a 3 on a die.
  • Compound Event: An event with more than one outcome. Example: Getting an even number on a die (i.e., 2, 4, or 6).
  • Mutually Exclusive Events: Two events that cannot happen at the same time. Example: Getting a head and a tail on a single coin toss.
  • Independent Events: Two events where the occurrence of one does not affect the other. Example: Tossing two coins.
  • Dependent Events: Two events where the occurrence of one affects the other. Example: Drawing cards from a deck without replacement.
  • Complementary Events: Two events that cover all possible outcomes of an experiment, such that one event happening means the other does not. Example: "It will rain" and "It will not rain" are complementary events.

  • Multiplication Rule: For two independent events, the probability of both events happening is the product of their individual probabilities.
    • P ( A and B ) = P ( A ) × P ( B )

    If A and B are independent events.

2. Bayes' Theorem

Definition

  • Bayes' Theorem is a fundamental concept in probability theory that describes how to update the probability of a hypothesis based on new evidence.
  • Formula: P A = P B|A P B

Formula for Bayes' Theorem

  • The formula for Bayes' Theorem is:
    • P A|B = P B|A ⋅ P A P B
  • Where:
    • P(A|B) is the posterior probability: the probability of event A occurring given that event B has occurred.
    • P(B|A) is the likelihood: the probability of event B occurring given that event A is true.
    • P(A) is the prior probability: the initial probability of event A before considering event B.
    • P(B) is the evidence or marginal likelihood: the total probability of event B occurring under all possible scenarios.

Steps to Use Bayes' Theorem

  • Identify the events: Determine what events A and B represent in your problem.
  • Calculate the prior probability: Find P(A), the probability of A occurring before considering new evidence.
  • Calculate the likelihood: Find P(B|A), the probability of observing new evidence B if A is true.
  • Calculate the evidence: Find P(B), the total probability of observing evidence B, considering all possible events.
  • Apply Bayes' Formula: Use the formula to update the probability of A given the new evidence B.

Example

Consider a factory producing two types of products:

  • Type A: 80% of production
  • Type B: 20% of production

We are given that the probability of a defective product is:

  • P(B|A) = 0.01 (probability that Type A is defective)
  • P(B|B) = 0.05 (probability that Type B is defective)

A product is randomly selected and found to be defective. What is the probability that it is Type A?

Solution:

  • Prior Probability: P(A) = 0.8 (since 80% of products are Type A)
  • Likelihood: P(B|A) = 0.01 (probability that Type A is defective)
  • Evidence: The total probability of a defective product is calculated as: P B = P B|A ⋅ P A + P B|B ⋅ P B = (0.01 * 0.8) + (0.05 * 0.2) = 0.008 + 0.01 = 0.018
  • Posterior Probability: Now apply Bayes' Theorem: P A|B = P B|A ⋅ P A P B = (0.01 * 0.8) / 0.018 ≈ 0.444 Thus, the probability that the defective product is Type A is approximately 44.4%.

Key Points of Bayes' Theorem

  • Prior Knowledge: Bayes' Theorem allows you to update the probability of an event based on new evidence.
  • Reversing Conditional Probabilities: It is useful when you want to find P(A|B) but know P(B|A).
  • Machine Learning: Bayes' Theorem is used in models like Naive Bayes for classification problems.
  • Real-Life Use: It is widely used in decision-making, especially when new evidence changes the likelihood of outcomes.

Applications of Bayes' Theorem

  • Medical Diagnosis: Updating the probability of a disease based on test results.
  • Spam Filtering: Classifying emails as spam or not based on their content.
  • Risk Assessment: Evaluating the likelihood of financial or insurance-related events occurring.
  • Machine Learning: Used in algorithms like Naive Bayes to classify data based on conditional probabilities.

3. Binomial Distribution

Definition

  • The Binomial Distribution is a probability distribution that models the number of successes in a fixed number of independent trials of a binary (yes/no) experiment.
  • Each trial has two possible outcomes: success or failure.

Key Characteristics

  • Binary Outcome: Each trial has two possible outcomes, often labeled "success" and "failure."
  • Fixed Number of Trials: The experiment is conducted a fixed number of times, denoted by n.
  • Constant Probability of Success: The probability of success on each trial, denoted by p, remains the same for each trial.
  • Independence: The trials are independent of each other, meaning the outcome of one trial does not affect the others.

Binomial Distribution Formula

The probability of observing exactly k successes in n independent trials is given by the binomial probability mass function (PMF):

P X = n k p k ( 1 - p n-k )

Where:

  • P(X = k): Probability of exactly k successes in n trials.
  • n: Total number of trials.
  • k: Number of successes.
  • p: Probability of success on a single trial.
  • 1 - p: Probability of failure on a single trial.
  • (n choose k): Binomial coefficient, the number of ways to choose k successes from n trials, calculated as: n ! k ! (n-k) !

Mean and Variance of Binomial Distribution

  • Mean (μ): μ = n ⋅ p
  • Variance (σ²): σ² = n ⋅ p ⋅ (1 - p)
  • Standard Deviation (σ): σ = √(n ⋅ p ⋅ (1 - p))

Example

Suppose a factory produces light bulbs, and each bulb has a 95% chance of being functional (success) and a 5% chance of being defective (failure). If 10 bulbs are tested, what is the probability that exactly 8 bulbs are functional?

Given:

  • n = 10 (number of trials)
  • p = 0.95 (probability of success)
  • k = 8 (number of successes)

We can use the binomial distribution formula:

P X = 10 8 0.95 8 ( 0.05 2 )

First, calculate the binomial coefficient:

(10 choose 8) = 10 ! 8 !( 10-8 )!

This simplifies to:

(10 choose 8) = 45

Now, plug in the values:

P X = 45 0.95 8 ( 0.05 2 )

Result:

P(X = 8) ≈ 0.0747 (7.47%)

Applications of Binomial Distribution

  • Quality Control: Used in manufacturing to model the number of defective items in a batch of products.
  • Surveying: In market research to determine the number of people who agree with a certain statement out of a sample of respondents.
  • Medical Studies: To model the number of patients who recover after treatment out of a sample.
  • Game Theory and Gambling: To calculate the likelihood of a certain number of wins in a series of games or bets.

Conditions for Using the Binomial Distribution

  • Fixed number of trials.
  • Each trial has only two possible outcomes (success or failure).
  • The trials are independent.
  • The probability of success is the same for each trial.

2. Trigonometric Functions, Inverse Trigonometric Functions, and Their Properties - Short Notes

i. Trigonometric Functions:

Trigonometric functions are functions of angles and are fundamental in the study of geometry, particularly in right triangles and circular motion.

Sine:

sin(θ) = opposite / hypotenuse

Cosine:

cos(θ) = adjacent / hypotenuse

Tangent:

tan(θ) = opposite / adjacent = sin(θ) / cos(θ)

Cosecant:

csc(θ) = 1 / sin(θ)

Secant:

sec(θ) = 1 / cos(θ)

Cotangent:

cot(θ) = 1 / tan(θ) = cos(θ) / sin(θ)

ii. Inverse Trigonometric Functions:

Inverse trigonometric functions are the inverse operations of the standard trigonometric functions. They give the angle corresponding to a given trigonometric value.

Inverse Sine (sin−1 x or arcsin x):

sin−1 x = θ if sin(θ) = x, −π/2 ≤ θ ≤ π/2

Inverse Cosine (cos−1 x or arccos x):

cos−1 x = θ if cos(θ) = x, 0 ≤ θ ≤ π

Inverse Tangent (tan−1 x or arctan x):

tan−1 x = θ if tan(θ) = x, −π/2 < θ < π/2

Inverse Cosecant (csc−1 x):

csc−1 x = θ if csc(θ) = x, −π/2 ≤ θ ≤ π/2, θ ≠ 0

Inverse Secant (sec−1 x):

sec−1 x = θ if sec(θ) = x, 0 ≤ θ ≤ π, θ ≠ π/2

Inverse Cotangent (cot−1 x):

cot−1 x = θ if cot(θ) = x, 0 < θ < π

iii. Properties of Trigonometric Functions:

Periodicity:

  • Sine and Cosine: Period 2π
  • Tangent and Cotangent: Period π
  • Secant and Cosecant: Period 2π

Range:

  • Sine, Cosine: [−1, 1]
  • Tangent, Cotangent: (−∞, ∞)
  • Secant, Cosecant: (−∞, −1] ∪ [1, ∞)

Symmetry:

  • Sine and Tangent: Odd functions, sin(−θ) = −sin(θ), tan(−θ) = −tan(θ)
  • Cosine, Secant, Cotangent: Even functions, cos(−θ) = cos(θ), sec(−θ) = sec(θ)

iv. Properties of Inverse Trigonometric Functions:

Range:

  • sin−1 x: [−π/2, π/2]
  • cos−1 x: [0, π]
  • tan−1 x: (−π/2, π/2)
  • csc−1 x: [−π/2, π/2] (excluding 0)
  • sec−1 x: [0, π] (excluding π/2)
  • cot−1 x: (0, π)

  • cos(cos−1 x) = x, for x ∈ [−1, 1]

3. Heights and Distances

Definition

  • Heights and Distances is a branch of Trigonometry used to calculate the height, distance, or angle of elevation/depression of objects using trigonometric principles.

Key Concepts

  • Angle of Elevation: The angle formed by the line of sight from the observer to the top of an object above the observer.
  • Angle of Depression: The angle formed by the line of sight from the observer to an object below the observer.

Basic Trigonometric Ratios

  • For a right triangle with angle θ, base b, height h, and hypotenuse l, the following relations are useful:
    • sin ⁡ θ = opposite hypotenuse
    • cos ⁡ θ = adjacent hypotenuse
    • tan ⁡ θ = opposite adjacent

Important Formulas

  • Height of an Object (using tanθ): When the distance from the observer to the object and the angle of elevation are known: h = d ⋅ tan ⁡ θ
  • Distance to an Object (using tanθ): When the height of the object and the angle of elevation are known: d = h tan ⁡ θ
  • Height of an Object (using sinθ or cosθ): If the distance to the object (or the length of the base) and the angle of elevation are known: h = l ⋅ sin ⁡ θ

Solving Problems in Heights and Distances

  • Example 1: A person standing at a distance of 30 m from a tower observes that the angle of elevation to the top of the tower is 30°. Find the height of the tower.
    • Using the formula: h = d ⋅ tan ⁡ 30°
    • Answer: The height of the tower is approximately 17.31 meters.
  • Example 2: A man is standing on the ground, looking up at the top of a building. The angle of elevation is 45°, and the height of the building is 50 meters. Find the distance between the man and the building.
    • Using the formula: d = h tan ⁡ 45°
    • Answer: The distance from the man to the building is 50 meters.

Applications of Heights and Distances

  • Navigation: Used to determine the distance between two points using angles of elevation or depression.
  • Architecture: Determining the height of buildings, towers, and other structures using measurements at ground level.
  • Surveying: Used to determine the height of mountains, bridges, and other large structures.
